Guide du banc d'essai

Les nouvelles machines Caterpillar utilisent les commandes électroniques, qui peuvent aider à identifier des modules, des composants et des modes de défaillance spécifiques.

Le code erreur est sous forme CID/FMI.

Vous trouvez ci-joint les codes erreurs dans ce projet avec l'explication de chaque code est un lien que vous permet de trouver la solution pour fixer l'erreur.
